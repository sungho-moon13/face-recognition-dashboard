# Face Recognition Dashboard

Project initialized based on [PRD.md](./PRD.md).

## Project Structure
- `src/`: Source code directory.
  - `src/camera/`: Camera capture logic.
  - `src/model/`: Face recognition model integration.
  - `src/web/`: Web application interface.
  - `src/utils/`: Utility functions.
- `docs/`: Project documentation.
- `tests/`: Unit and integration tests.
- `PROJECT_PLAN.md`: Detailed project plan and GitHub milestones.

## Usage
1.  Install dependencies: `pip install -r requirements.txt`
2.  Run the application: `python src/web/app.py`
